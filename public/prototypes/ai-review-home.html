전체 프로젝트

프로그램 설명

  1. 01

    리뷰 수집·정제

    Steam에서 P의 거짓 한국어 리뷰 1,315건을 수집했습니다. 내용이 지나치게 짧거나 분석하기 어려운 리뷰를 제외하고, 최종 1,059건을 분석에 사용했습니다.

  2. 02

    API 수집 및 LLM 분석

    Steam API로 리뷰를 수집하고, OpenRouter를 통해 LLM에 분석을 요청했습니다.

    감성
    긍정·부정·혼합·중립
    감정
    만족·기쁨·분노·실망·중립
    게임 영역
    전투·성능 및 버그·스토리·그래픽·가격
    분석 근거
    핵심 문장과 판단 신뢰도
  3. 03

    실제 반응에 맞게 결과 조정

    개선점을 충분히 찾기 위해 비추천 리뷰를 의도적으로 더 많이 포함했습니다. 실제보다 부정적으로 보이지 않도록 Steam 전체 추천 비율을 기준으로 결과를 다시 맞췄습니다.

    수집 데이터 91%→ 보정 후 92%→ Steam 실제 92%
  4. 04

    현재 확인 가능한 기능

    실시간 리뷰 수집과 신규 분석에는 API 키와 별도 서버가 필요해 현재 화면에서는 제외했습니다. 대신 실제 분석으로 완성한 P의 거짓 리뷰 1,059건과 결과 대시보드를 제공합니다.

    직접 확인 가능: 리뷰 탐색, 필터링, 주요 인사이트와 보정 결과

새 분석은 준비 중입니다

분석이 완료된 프로젝트를 열어 대시보드와 리뷰 데이터를 직접 탐색할 수 있습니다.
